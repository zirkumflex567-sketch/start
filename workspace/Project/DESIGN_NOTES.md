# Design Notes

## Drift Phases
- Pre-drift: subtle foreshadowing cues (audio hush, palette shift).
- Onset: gentle acceleration, direction flip, or time dilation.
- Reverse: calm return to the baseline pace.
- Afterglow: lingering ambience and soft visual bloom.

## Encounter Categories
- Wildlife: birds, deer, drifting sprites.
- Wayfarers: NPCs with quiet gestures or signs.
- Moments: weather shifts, ruins, ambient props.
- Treasures: memory fragments, cosmetic unlocks, archive tags.

## UI Tone Notes
- Whispered prompts, minimal text.
- Rounded, soft UI elements with generous spacing.
- Audio-first feedback whenever possible.
